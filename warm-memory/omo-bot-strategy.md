# omo.bot 오픈 이후 전략
> 작성: 2026-03-08 | 대표님 요청 | 지노 분석

---

## 서비스 요약 (현황 파악)
- **타겟**: 비개발자 한국인 (1분 설정, 코딩 불필요)
- **채널**: 텔레그램 기반 AI 에이전트
- **모델**: Free (기본) + 토큰 충전 (프론티어 모델)
- **현재**: Closed Beta 100명 → 대기리스트

---

## 1. 마케팅 전략

### 🎯 핵심 포지셔닝
> "해외 AI는 비싸고, OpenClaw는 어렵다. omo.bot은 1분이다."

비교 테이블이 이미 랜딩에 있음 → 경쟁자 명확. 이 포지셔닝을 마케팅에 직접 활용.

---

### Phase 1: 오픈 직후 (1~2주) — 화제성 확보

**바이럴 훅**
- "텔레그램으로 쿠팡 최저가 주문하는 AI 만들어봤다" 형태의 짧은 영상 콘텐츠
- 실사용 데모 GIF/릴스: 배달주문, 일정잡기, 환율조회 등 일상 시나리오
- X(트위터) + 스레드 + 인스타에 동시 배포

**초기 사용자 활용**
- 베타 100명 → 후기 적극 수집, 스크린샷 공유 유도
- "나만의 오모봇 이름이 뭐예요?" 해시태그 캠페인 (개인화 강조)
- 베타 유저 레퍼럴: 친구 초대 시 토큰 보상

**채널별 역할**
| 채널 | 목적 | 콘텐츠 형태 |
|------|------|-------------|
| X (트위터) | 화제성, 개발자 커뮤니티 | 데모 영상, 비교 짤 |
| 인스타/릴스 | 비개발자, MZ 타겟 | 일상 시나리오 영상 |
| 유튜브 쇼츠 | 유입 최대화 | "AI로 배달시켜봤다" 형 |
| 블로그/브런치 | SEO, 설명글 | "omo.bot이 뭔지 5분 만에" |
| 커뮤니티 | 직접 설명 | 클리앙, 뽐뿌, 직장인 커뮤니티 |

**KPI**
- 대기리스트 등록 수 (1차 목표: 1,000명)
- 텔레그램 채널/그룹 가입자
- 데모 영상 조회수

---

### Phase 2: 본 오픈 후 (1~3개월) — 사용자 정착 & 수익화

**콘텐츠 마케팅**
- "오모봇 활용법 시리즈": 쇼핑편, 일정편, 금융편 등 카테고리별
- 사용자 스토리: 실제 유저가 뭘 했는지 인터뷰/소개
- "이번 달 오모봇으로 얼마 아꼈어요" 후기 캠페인

**B2B 잠재 발굴**
- 1인 사업자/프리랜서: 일정·이메일 관리 니즈
- 소상공인: 배달·재고 알림 등
- 시니어: 가족이 세팅해주는 "부모님 AI"

**토큰 수익화 트리거**
- Free → 유료 전환 유도: "GPT-5.4로 해보세요" 인앱 프롬프트
- 첫 충전 할인 이벤트 (50% 보너스 토큰)

## 2. 커뮤니티 운영 방식

### ✅ 결론: 텔레그램 메인, 디스코드 서브 (개발자용)

**이유**
- 서비스 자체가 텔레그램 기반 → 사용자가 이미 텔레그램에 있음
- 비개발자 타겟 → 디스코드 진입장벽 높음
- 텔레그램 그룹/채널 기능이 충분히 강력

---

### 텔레그램 구조 (추천)

```
📢 omo.bot 공지채널 (읽기 전용)
   └ 업데이트, 장애공지, 이벤트

💬 omo.bot 사용자 모임 (그룹)
   └ 자유 대화, 활용법 공유, Q&A

🤖 omo.bot 피드백 봇 (자동화)
   └ 버그 리포트, 기능 요청 접수
```

**운영 방식**
- 공지채널: 업데이트마다 공식 발표
- 사용자 그룹: 활용 팁 공유, 질문 응답 (초기엔 운영자 직접 참여)
- 주 1회 "이번 주 오모봇 하이라이트" 요약 포스팅
- 적극 유저 → "오모봇 파워유저" 뱃지/역할 부여

**디스코드 (선택적 운영)**
- 개발자/얼리어답터 대상으로만 운영
- MCP 확장 논의, API 연동 피드백
- GitHub 연동, 기술 피드백 채널

---

### 커뮤니티 성장 전략
- 초기: 오너가 직접 답변 (진정성 극대화)
- 파워유저 선발 → 커뮤니티 매니저로 전환
- 월 1회 "오모봇과의 AMA" (대표님 직접 참여)
- 우수 활용사례 → 랜딩페이지 소개 (유저 보람)

### 확장 로드맵 (추천 순서)

**🥇 1순위: 한국 필수 인프라 (1~2개월)**
1. **카카오 통합** — 카카오맵(길찾기), 카카오T(이미 있지만 강화), 카카오 알림톡
2. **네이버페이 / 토스** — 결제 + 잔액 조회 → 금융 기능 완성
3. **네이버 예약** — 식당/헬스장/병원 예약 자동화 (일상 최고 니즈)

**🥈 2순위: 생활 밀착 서비스 (2~3개월)**
4. **당근마켓** — 중고거래 알림, 가격 조회
5. **유튜브** — 콘텐츠 추천, 요약 (엔터테인먼트 진입)
6. **Notion / 구글 드라이브** — 직장인/학생 타겟 확장

**🥉 3순위: 전문화 (3~6개월)**
7. **스마트스토어** — 셀러 타겟 (B2B 수익 확장)
8. **건강 데이터** — 걸음수, 수면 분석 (웰니스 시장)
9. **해외 서비스** — Airbnb, Booking.com (여행 예약 완성)

---

### MCP 확장 원칙
1. **한국 서비스 우선** — 글로벌 툴은 이미 대안 많음. 한국 서비스 연동이 차별점
2. **일상 빈도 기준** — 주 3회 이상 쓸 서비스만 우선 검토
3. **인증 난이도 체크** — Open API 있는 곳 먼저, 스크래핑은 나중
4. **유저 요청 데이터 활용** — 커뮤니티 피드백 → 우선순위 실시간 반영
