## 한줄 요약
팜모닝 플랫폼을 통해 14억 농부의 데이터 기반 영농을 지원하는 한국 애그테크 선도 기업

## 핵심 포인트
- 팜모닝: 생산~유통 데이터 기반 영농 관리 플랫폼
- 글로벌 1위 애그테크 비전, 잠재 대상 14억 농부
- 농업 디지털화 선도 (전통적 변화 저항 산업)
- 한국 내 강력한 기반, 글로벌 확장 계획
- 데이터 네트워크 효과: 농부 증가 → 데이터 품질 향상
- 생산→유통 풀스택 플랫폼 접근
- 수익화 모델 및 단위경제 업데이트 필요
- 글로벌 확장 현황 모니터링

## 전문 요약
GreenLabs는 팜모닝 플랫폼을 통해 데이터 기반 영농을 지원하는 한국 애그테크 기업이다. 생산부터 유통까지 풀스택 솔루션을 제공하며 14억 글로벌 농부를 잠재 대상으로 한다. 데이터 네트워크 효과와 풀스택 접근 방식을 통해 농업 디지털화를 선도하고 있다.
