PortOne(구 CHAI)은 아시아 전역 상인과 플랫폼을 위한 API 기반 결제 인프라 솔루션이다. Daniel Shin(Terraform Labs 공동창업자)이 세웠으며 최근 1.53M 다운라운드는 밸류에이션 압박 신호다. 결제 인프라 네트워크 효과는 강력하나 테라폼 연관 이미지 리스크 관리가 과제다.
