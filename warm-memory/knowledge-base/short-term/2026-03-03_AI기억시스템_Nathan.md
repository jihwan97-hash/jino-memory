## 한줄 요약
지식을 AI의 도움으로 데이터베이스화하고 단기→중기→장기 기억으로 점차 증류해 저장하면 정보 홍수 속에서 효율적으로 배울 수 있다.

## 핵심 포인트
- 새 지식을 마주할 때 판단 기준은 사전 지식 + 가치관 두 가지
- 링크/아티클을 에이전트에 보내면 기존 지식과 비교해 새롭고 유용한 부분만 추출
- 저장 형식: 해시태그(인덱스) + 짧은 요약 + Bullet 10개 + 소스 & 전문
- 이미 아는 내용이 들어오면 피드백 → 모델이 원하는 지식 유형 학습
- 단기→중기 전환 시 휘발 위험 높으므로 자동 복습 시스템 필수
- 해시태그로 관련 지식 불러와 AI가 복습 유도
- 비개발자도 AI 에이전트로 이 시스템 구축 가능
- @simonkim_nft 의 기억 프레임워크 + @dan_contxtsai 방법론 벤치마크
- OpenClaw 에이전트로 실제 구현한 사례
- 잘 셋팅하면 "딸깍"으로 자동 작동

## 전문 요약
AI 관련 정보가 빛의 속도로 생산되는 시대에 피로 없이 따라가려면 지식 습득 방식을 잘 셋팅해야 한다. 새 지식 소스(X, 기사, 유튜브)를 발견하면 에이전트에 전송 → 기존 지식 베이스와 비교해 새롭거나 유용하거나 흥미로운 부분만 추출·요약 → 해시태그+요약+불릿+원문 형식으로 저장. 원치 않는 지식에 피드백을 주면 모델이 선호를 학습한다. 단기기억이 중기기억으로 넘어갈 때 해당 해시태그로 지식을 불러와 AI가 자동 복습을 유도한다.
