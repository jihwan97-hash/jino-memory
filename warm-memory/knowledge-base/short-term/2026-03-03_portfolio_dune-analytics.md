## 한줄 요약
100개 이상 블록체인 SQL 기반 온체인 데이터 분석 플랫폼, 2026년 2월 스테이블코인 데이터셋 출시

## 핵심 포인트
- 100개+ 블록체인 SQL 쿼리 기반 데이터 분석 플랫폼
- 연구자, 트레이더, 프로토콜, 기관 모두 사용하는 산업 표준
- 2026.02 스테이블코인 통합 데이터셋 출시 (EVM, Solana, Tron)
- API & DataShare로 프로그래매틱 데이터 스트리밍
- 무료 플랜 + 기업급 API 구조
- HVF 최우선 포트폴리오 태깅
- 지속적 제품 혁신으로 PMF 입증

## 전문 요약
Dune Analytics는 SQL을 통해 100개 이상의 블록체인 데이터를 직접 쿼리하고 시각화할 수 있는 업계 표준 블록체인 분석 플랫폼이다. 2026년 2월 스테이블코인 통합 데이터셋을 출시하며 지속적인 제품 혁신을 보여주고 있다. HVF 포트폴리오 중 최우선 순위 기업.
