## 핵심 포인트
- Ferum Protocol: Aptos 기반 온체인 CLOB
- 자산 불가지론적(asset-agnostic) 오더북 구조
- Move 언어(메모리 안전) 기반 스마트 컨트랙트
- 유동성 공급자에게 전례 없는 통제권 부여
- AMM 방식 대신 중앙화 거래소급 오더북 제공
- GitHub 오픈소스 (ferumlabs/ferum)
- 최근 자금 조달 및 TVL 정보 업데이트 필요
- Aptos 생태계 성장과 연동된 성과

## 전문 요약
Ferum은 Dolores Park Labs가 개발한 Aptos 블록체인 위의 온체인 CLOB(중앙집중형 리밋 오더북) DeFi 인프라다. Move 언어의 메모리 안전성과 Aptos의 고성능을 결합해 AMM 방식을 대체하는 기관급 트레이딩 인프라를 제공한다.
