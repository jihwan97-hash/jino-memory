# Knowledge Base — 지식 파이프라인

## 개요
새로운 지식(링크, 아티클, 유튜브)을 AI의 도움을 받아 데이터베이스화하고,
단기기억 → 중기기억 → 장기기억 형태로 점차 증류하며 저장.

## 저장 구조

### 단기기억 (`short-term/`)
- 최근 7일 내 수집된 지식
- 형식: `YYYY-MM-DD_제목.md`
- 자동 저장, 빠른 훑기용

### 중기기억 (`mid-term/`)
- 단기에서 1주일 후 리뷰 → 가치있으면 승격
- 주제별 통합, 복습 유도

### 장기기억 (`long-term/`)
- 중기에서 한 달 후 리뷰 → 핵심만 증류
- MEMORY.md에도 반영

## 저장 형식 (모든 항목 공통)

```markdown
# [제목]

**태그:** #태그1 #태그2 #태그3
**소스:** [URL]
**저장일:** YYYY-MM-DD
**티어:** short-term

## 한줄 요약
한 문장으로 핵심 정리.

## 핵심 포인트
- 포인트 1
- 포인트 2
- 포인트 3
- (최대 10개)

## 전문/원문 요약
원문의 주요 내용 전체 요약.
```

## 워크플로우

1. **수집**: 형이 링크/아티클 공유 → 지노가 브라우저로 읽기
2. **필터링**: 기존 지식 베이스와 비교 → 새롭거나 유용한 부분만 추출
3. **저장**: `short-term/`에 위 형식으로 저장 + `index.json` 업데이트
4. **주간 복습**: 단기 → 중기 승격 리뷰 (brain-memory 크론 활용)
5. **월간 증류**: 중기 → 장기 승격 + MEMORY.md 반영
