## Technical Architecture

### Based Contestable Rollup (BCR)
1. **Based Sequencing:**
   - Ethereum L1 validators order Taiko transactions
   - No MEV capture by separate sequencer
   - Inherits L1 liveness + censorship resistance

2. **Multi-Proof System:**
   - Configurable proof schemes
   - Contestation mechanism for invalid proofs
   - Security > speed trade-off

3. **Ethereum Equivalence:**
   - Full EVM compatibility (not just "compatible")
   - Direct tooling/infra portability

### Testnet History
- **Katla Testnet:** 6th testnet, first with BCR design (early 2024)
- Foundation for mainnet launch
