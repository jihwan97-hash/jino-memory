# Sejinsa (세진사)
Last researched: 2026-03-01

## Overview
- Business: 구독형 출장세차 서비스
- Tagline: "세차에 진심인 사람들, 믿을 수 있는 출장세차"
- HVF stake: 22.1%
- Korea-based subscription car wash service
