- Word .docx를 **XML 레벨에서 직접 수정** → 트랙 체인지 적용, 스타일 깨짐 없음
- Bluebook 인용 형식 **코드로 자동 재포맷** (수초 만에)

## 인사이트

### 판단력 인코딩 = 진짜 레버리지
- 기존 법률 AI(전문 SaaS): 도구 능력이 퀄리티 상한선
- Claude + 개인 프레임워크: 상한선이 **사용자의 판단력 자체**
- "회사 플레이북 → 개인 플레이북" 전환이 핵심

### Nathan 트윗과 같은 맥락 (delegate tax)
- 조직의 위임 세금(맥락 희석) 없이 판단력이 직접 전달됨
- 상위 실무자가 팀 전체 작업을 직접 처리 가능

### VC/투자 적용 가능성
- 투자 심사 프레임워크 인코딩 → 계약서/텀시트 리뷰 자동화
- 포트폴리오 모니터링 판단 기준 인코딩
- 딜 스크리닝 체크리스트 자동 적용
