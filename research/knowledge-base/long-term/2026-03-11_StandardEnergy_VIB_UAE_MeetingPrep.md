## TL;DR

> AWS DC가 이란 드론에 피격 → 화재→침수→서비스 중단. **"화재 0% ESS"가 nice-to-have → 안보 인프라로 격상.** VIB의 UAE 진입 타이밍이 이보다 좋을 수 없음.

## 1. 왜 UAE, 왜 지금

**수요**
- Stargate UAE 5GW (Phase 1 200MW, Q3 가동)
- MS × G42 200MW + Khazna Ajman 100MW
- 총 파이프라인 **6GW+**

**가격 시그널**
- EWEC 피크요금 **30% 인상** → DC peak-shaving ESS 수요 직접 발생

**기후**
- 여름 50°C → Li-ion 수명 3~4년 급감
- VIB **15~20년 무교체** → TCO 압도적 우위

**전쟁 (신규)**
- AWS DC 2곳 드론 직격 → 화재 → 소방 전력차단 → 수해(4cm) → 냉각 고장 → 수일간 서비스 중단
- **DC가 군사 타겟이 된 사상 최초 사례**
- VIB였다면 화재 연쇄 피해 자체가 없었음

## 2. 타겟 고객

### Tier 1 — Khazna (G42)
- G42(아부다비 AI·기술 홀딩, Tahnoun bin Zayed 회장) 60% + e&(구 Etisalat) 40%
- UAE 최대 DC 플랫폼, 시장점유율 **70%+**
- 아부다비·두바이·아즈만 **29개 DC** 운영
- Stargate UAE(G42+OpenAI+Oracle+NVIDIA+SoftBank, 5GW 캠퍼스 내 1GW 클러스터) 실제 건설 주체

### Tier 2 — EWEC (Emirates Water and Electricity Company)
- ADPower(아부다비 전력공사) 산하, 아부다비 에미리트 유일 전력·수자원 도매 조달 주체
- **17.9GW** 발전용량, 16개 발전소 운영
- 피크요금 30% 인상 주도 주체
- Masdar/EWEC 19GWh BESS($6B, 2027)는 CATL 확정 → 직접 경쟁 어렵지만 분산형 ESS 정책 벤치마크 가능

### Tier 3 — 한전 패키지 번들
- 한전 해외사업 3대 중점국: 사우디·UAE·호주
- 핵심: **타윌라C** — 아부다비 북동 50km, 2.5GW CCGT IPP
  - 한국중부발전+스미토모 / 한국서부발전+Etihad W&E+큐덴 입찰 참여 중
  - 향후 **7~8GW 추가 발주** 예상
- VIB를 한전 패키지 ESS 컴포넌트로 번들하는 중기 경로

## 3. VIB vs Li-ion (UAE 환경)

| 항목 | Li-ion | VIB |
|-----|-------|-----|
| 화재 | Thermal runaway 확률 증가 | **0%** (수계 전해질) |
| 수명 | **3~4년** (고온) | **15~20년** |
| 효율 | 85~92% | **96%** |
| 냉각 | HVAC 필수 | 불필요 |
| 단가 | 1x | ~2x |
| **15년 TCO** | **4~5x** | **1.3~1.5x** |

> 전후 보험료 상승 반영 시 TCO 격차 더 확대

## 5. 진입 경로

- **Path A — DC 내부 UPS 대체** (단기, 최현실적)
  - UPS 시장 $56M. 5C 출력으로 bridging + peak-shaving
  - 화재 0%로 규제 통과 우위. 피격 시 연쇄 피해 없는 UPS = 안보 요건

- **Path B — Behind-the-meter mid-scale ESS** (중기)
  - 1~50MWh. EWEC 피크요금 대응
  - 냉각 불필요 → 도심 설치 가능

- **Path C — 한전 에너지 패키지 번들** (중기)
  - 타윌라C + 추가 7~8GW. ESS를 한전 인프라에 번들
