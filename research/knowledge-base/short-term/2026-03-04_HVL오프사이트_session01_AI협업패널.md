## 한줄 요약
AI 시대의 협업은 "분업 후 합치기"가 아니라 각자 풀스택으로 처리 후 컨텍스트를 공유하는 방식으로 전환되고 있으며, 거버넌스의 핵심은 reversibility(복구 가능성)를 기준으로 권한을 위임하는 것이다.

## 핵심 포인트
- 1인 멀티 펑셔널리 시대: 한 사람이 디자인·개발·기획·QA 모두 처리 가능 → 역설적으로 인간 싱크(동기화)가 병목
- PR 패러다임 전환: merge conflict 해결 대신 커밋 로그를 컨텍스트로 삼아 처음부터 재개발하는 방식이 효율적
- 하용호 24시간 라이프로깅: Apple Watch + 맥북 화면 녹화 + 음성 → Obsidian 벡터라이즈로 인간 단기 기억 한계 극복
- 여준호 QMD 시스템: 벡터 서치 → 리랭크(최신성/적합성) → 자동 컨텍스트 주입으로 RAG 품질 향상
- 오라클 에이전트 패턴: 여러 에이전트가 독립 검증 → 투표로 합의 결과 채택, 단일 LLM 편향 상쇄
- Git Worktree: 에이전틱 시대 형상관리 베스트 프랙티스 — 여러 에이전트가 병렬로 다른 브랜치에서 작업
- 권한 위임 기준: AI가 책임질 수 없는 영역(비가역적 결과)에는 권한 부여 안 함
- Borg 백업 시스템: 증분 백업 10초 완료 → "망해도 10초면 복구"가 에이전트에게 과감한 자유를 주는 근거
- 프롬프트 인젝션 보안: 3시간 레드팀 훈련 실시 → 보안 가이드라인 수립, AI 권한 확대와 함께 보안 중요성 증가
- 팀 레포 공유 시스템: 공통 Claude 지시문·비즈니스 지식을 쿠버네티스 팟 생성 시 자동 로드 → AI-native 조직 지식 전달

## 전문 요약
HVL 오프사이트 1번 세션. 김민수 발표 + 패널 토론 형식으로 AI 시대의 협업·평가·거버넌스를 다뤘다.

**협업**: 1인이 풀스택을 처리할 수 있게 되면서 "분업 후 합치기" 패러다임이 붕괴. PR 충돌 해결보다 커밋 히스토리를 컨텍스트로 삼아 재개발이 빠른 사례가 늘어남. 하용호는 24시간 라이프로깅(Apple Watch·맥북 화면·음성)으로 인간 기억 한계를 극복하는 시스템을, 여준호는 QMD(Query-Match-Deliver) 시스템으로 자동 컨텍스트 주입을 구현.

**평가**: 오라클 에이전트 패턴(여러 에이전트가 독립 검증 후 투표)으로 단일 LLM 편향 상쇄. 픽셀 퍼펙트 클론 코딩은 픽셀 dist로 정량 평가. "벤치마크 점수와 실무 역량은 별개" — 실전 배치 후 피드백 루프가 더 중요.

**거버넌스**: 결과의 reversibility가 권한 위임의 핵심 기준. 샌드박스에서 검증 후 프로덕션 적용. 김연규의 오픈플로우 사례(1Password 공유·연말정산·쿠팡 주문 AI 대행)는 극단적 위임 사례. Borg 백업(증분 10초)이 실험적 위임을 가능하게 하는 인프라적 근거.
