# BTS 광화문 공연 데이터 분석
**출처**: 아이지에이웍스 (iGaWorks) 블로그  
**원문**: https://www.igaworksblog.com/post/insight-bts-fandom-audience  
**공연일**: 2026년 3월 21일  
**저장일**: 2026-03-31

---

## 규모

- 광화문 현장 방문자 **~16만 명** (공연 시간대 8~9시 ~5만 명)
- 넷플릭스 라이브 시청자 **~118만 명**
- 넷플릭스 DAU 전일 대비 **+56.6%**, 신규 설치 **+353.8%**
- 현장 방문자 중 **25.4% (4.2만 명)**이 지방에서 원정

---

## 팬덤 세그먼테이션 방법론

- 단순 넷플릭스 시청만으로는 우연 시청자 vs 진성 팬 구분 불가
- **위버스(월 10회+) × NOL티켓(티케팅 앱)** 데이터 결합해 노이즈 제거
- **GPS × 앱 × 결제 데이터 3중 결합** → 시공간 발자국을 광고 타겟 오디언스로 전환

---

## 오프라인 슈퍼 팬덤 (5.1만 명)

- 남성 비중 **40.8%** (예상 대비 높음 — 동반인·축제 유동인구 포함)
- 간편결제 일반 대비 **+104%**, 패션/의류 **+19%**
- 현장에서 즉각 소비하는 **"실행력 있는 소비자"**
- 게임 카테고리 **+26%** → IP 기반 게임 콜라보 타겟 가능성

---

## 온라인 슈퍼 팬덤 (1.8만 명)

- 여성 **93%**, 20~40대 **80%**
- SNS/커뮤니티 사용 일반 대비 **+206%**, 음악 **+86%**, OTT 앱 수 **+115%**
- 지방 거주 비중 **38%** (오프라인 28% 대비 +10%p) → 현장 못 오는 대신 굿즈·디지털 콘텐츠에 소비
- **"바이럴 엔진"** 역할

---

## 전체 슈퍼 팬덤 소비 프로파일

| 지표 | 수치 |
|------|------|
| 월 OTT 시청 | 53시간 |
| 여행/교통 앱 실행률 | 92.8% → 목적형 이동 라이프스타일 |
| 쇼핑 앱 체류 | 월 5.2시간 |
| 배달 앱 주문 | 3명 중 1명이 월 10회+ |
| SNS 사용 | 평균 3개+ → 정보 실시간 확산 구조 |

---

## 5대 타겟 마케팅 시나리오

1. **뷰티/F&B**: 오프라인 슈퍼 팬덤 대상, 공연 후 피로 보상 심리 리타게팅
2. **숙박/여행**: 지방 원정 4.2만 명 대상, 차기 공연 숙박 패키지 락인
3. **커머스/굿즈**: 온라인 슈퍼 팬덤 1.8만 명 대상, 단독 굿즈·VOD
4. **패션/OTT**: 라이트 팬덤의 바이럴 파워 활용, SNS 시크릿 링크 한정판
5. **팬덤 플랫폼**: 넷플릭스 신규 설치 + 위버스 미사용 ~100만 명 → 헤비 팬덤 전환 유도

---

## 투자 관점 시사점

- **GPS × 앱 × 결제 데이터 결합 SCI 방법론** = K-엔터 IP 마케팅 인프라 레이어
- K-팝 포트폴리오사 평가 시 **"데이터 기반 팬덤 세그멘테이션 역량"** 유무가 밸류에이션 차별화
- 지방 원정 25.4% 데이터 → 콘서트·팝업 비즈니스의 **숙박/교통 번들링 수익 모델** 정량 근거
- 오프라인·온라인 팬덤의 소비 패턴 차이 → **IP 콜라보 설계 시 채널별 전략 분리** 필요
