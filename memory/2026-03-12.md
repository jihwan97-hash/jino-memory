### 비상장 장외거래소 지분 규제 정리

**확인된 사실:**
- 자본시장법 시행령 **제7조의3 제2항**: ATS 거래량 15% 상한 (대주주 지분이 아닌 **거래량 점유율** 상한)
- 가상자산거래소 대주주 **지분 15~20% 제한**: 디지털자산기본법 2단계에서 *입법 추진 중* — 아직 법제화 전
- 금융위 논리: "인가제 전환 = 금융 인프라 = ATS 준용" → 15% 지분 상한 확산 시도
- 비상장 장외거래소 인가 요건 (2025.09 자본시장법 시행령): 자기자본 60억/인력 요건 명시, 대주주 지분 상한은 **명시 없음**
- 서울거래 50%/33% 수치: 공개 법령이나 뉴스에서 확인되지 않음 — 외부 검증 불가

**잘못 썼던 것 (교훈):**
- "넥스트레이드 선례 50%/33%" → 확인 안 됨, 내 추론이었음
- "상법·자본시장법 일반 원칙으로 금융 인프라에 일반 적용" → 확인 안 됨
- ATS 15%룰을 "대주주 지분 제한"으로 잘못 설명 → 실제로는 거래량 상한
