# CBT Mode — Reveal 의도 재정렬 스킬

> "지금 내 결과를 보면, 나의 진짜 의도를 알 수 있다."

---

## 핵심 공식

```
결과(Result) = 의도(Intention) × 방법(Mechanism)
```

**핵심 통찰**: 결과가 나쁜 건 방법이 없어서가 아니다. 의도(특히 숨은 의도)가 표면 의도를 상쇄하고 있기 때문이다. 의도가 마이너스면 방법을 백 개 알아도 쓸모없다.

---

## 의도의 이중 구조

```
표면 의도 (Stated):  "나는 ~을 원한다" — 의식적, 내가 아는 것
숨은 의도 (Hidden):  "사실은 ~이 두렵다" — 무의식적, 내가 모르는 것

결과 = 표면 의도 + 숨은 의도의 총합 (net)
```

**수식 예시** (어머니에게 전화하지 않는 경우):
```
표면 의도: +80 (관계 개선)
숨은 의도 1: -30 (거절 공포)
숨은 의도 2: -40 (복수심)
숨은 의도 3: -20 (이미지 보호)

순 의도(Net Intention) = -10 → 전화 안 함
```

**표류(Drifting)**: 의지력 부족이 아니라 숨은 의도가 표면 의도를 상쇄하는 신호다.
- "왜인지는 모르겠지만 계속 안 하게 된다"
- "시작은 하는데 끝까지 못 간다"
- "뭔가 막혀있는 느낌인데 뭔지 모르겠다"

---

## 4가지 숨은 의도 드라이버

### Being Right — "내가 옳다"
- 핵심: 틀리지 않고 싶다. 내 판단이 맞다는 걸 증명하고 싶다
- 신호: 피드백을 위협으로 받아들임 / 논쟁에서 이기려 함 / "나는 이미 알고 있어"

### Being Comfortable — "편하고 싶다"
- 핵심: 불편함/갈등을 회피하고 싶다. 현상 유지가 안전하다
- 신호: "나중에 하지 뭐" / 어려운 대화를 미룸 / 변화 필요성을 알면서도 익숙함에 머무름

### Being in Control — "통제하고 싶다"
- 핵심: 결과를 예측하고 조종하고 싶다. 불확실성이 불안하다
- 신호: 완벽한 계획이 나올 때까지 대기 / 위임 못 함 / "준비가 더 필요해"

### Looking Good — "있어 보이고 싶다"
- 핵심: 무능해 보이기 싫다. 타인의 평가로부터 이미지를 보호하고 싶다
- 신호: 모른다고 말 못 함 / 실패할 수 있는 일을 공개적으로 시도하지 않음 / 도움 요청이 어려움

## 7단계 의도 재정렬 프로세스

### Step 1: 결과 직시 (Face the Result)
결과는 거짓말하지 않는다. 결과가 진짜 의도를 말해준다.

**핵심 질문**:
- 내가 "원한다"고 말하는 것과 실제 일어나고 있는 것의 간극은?
- 행동만 보고 추측한다면, 이 사람은 무엇을 원한다고 말할까?

### Step 2: 숨은 의도 인지 (Recognize Hidden Intentions)
4가지 드라이버 점검:

| 드라이버 | 질문 | 해당? |
|---------|------|------|
| Being Right | 이 상황에서 내가 "맞다"고 증명하려는 것은? | |
| Comfortable | 이 상황에서 내가 피하고 있는 불편함은? | |
| Control | 이 상황에서 내가 통제하려는 것은? | |
| Looking Good | 이 상황에서 내가 보호하려는 이미지는? | |

> 숨은 의도를 인지하는 것 자체가 변화의 절반이다. 무의식에서 의식으로 올라오면 "선택 가능한 것"이 된다.

### Step 3: 대가 계산 (Count the Price)
모든 숨은 의도에는 보상(Payoff)과 대가(Price)가 있다.

**핵심 질문**: "이 대가를 계속 지불할 것인가?"

### Step 4: 비전 재설정 (Reset the Vision)
표면적 원함을 관통하는 가장 깊은 레이어의 욕구 = 비전.

- "복수하고 싶다" → 방어 기제
- "거절당하기 싫다" → 방어 기제
- "어머니에게 사랑받고 싶다" → 이것이 비전

### Step 5: 의도 깎아내기 (Prune Intentions)
숨은 의도를 없애는 게 아니라, 비전이 더 크다는 걸 선택하는 것이다.

```
깎기 전:  복수심 40% + 거절공포 30% + 이미지보호 20% + 비전 10% = 전화 안 함
깎은 후:  비전 100% (나머지는 인지하지만 선택하지 않음) = 전화함
```

### Step 6: 스탠드 (Stand on Vision)
결과와 무관한 존재적 선언. 조건부 결심이 아니다.

```
"나는 [비전] 위에 선다. 결과와 상관없이.
[두려움]이 있어도, [불편함]이 있어도,
나는 이 비전 위에 선다."
```

스탠드의 기준은 "결과가 좋았느냐"가 아니라 "비전대로 살았느냐"다.

### Step 7: 시도 (Act — Mechanism Doesn't Matter)
의도가 100%가 되면 시도가 자연스럽게 발생한다. 방법은 뛰어든 후에 찾는다.

```
의도 100% → 시도 → 실패 → 다른 방법 → 실패 → 개선 → 결과
```

> "사랑하는 아이가 물에 빠졌을 때, 수영 방법을 모른다고 고민하지 않는다. 뛰어든다."

## 세션 진행 방식

숨은 의도 탐색은 소크라테스식 대화로 진행한다. 직접 결론을 말하지 않고 질문으로 스스로 발견하게 유도한다.

### 유용한 질문들
```
"지금 실제로 일어나고 있는 결과는 무엇인가요?"
"만약 행동만 보고 의도를 추측한다면?"
"이 상황에서 편안하게 피하고 있는 것이 있다면?"
"결과를 통제하려는 욕구가 행동을 막고 있지는 않나요?"
"이 대가를 계속 지불할 건가요?"
"이 모든 것 중에서, 진짜로 가장 원하는 것은 무엇인가요?"
```

## 자기 점검 워크시트

```
영역: ______________________

1. 결과 직시:    지금 실제 결과는?
2. 표면 의도:    나는 무엇을 원한다고 말하는가?
3. 간극:         둘 사이의 간극은?
4. 숨은 의도:    Being Right? Comfortable? Control? Looking Good?
5. 대가:         이 숨은 의도를 유지하는 대가는?
6. 비전:         진짜로 가장 원하는 것은?
7. 스탠드:       "나는 ________ 위에 선다. 결과와 상관없이."
8. 첫 시도:      지금 할 수 있는 가장 작은 시도는?
```

## 핵심 역설

| 역설 | 내용 |
|------|------|
| 방법이 상관없다 → 더 좋은 방법을 찾게 된다 | 완벽주의 해제 → 시도 증가 → 학습 증가 |
| 실패를 허용하면 → 실패가 줄어든다 | 공포 제거 → 순 의도 상승 → 성공 확률 상승 |
| 통제를 내려놓으면 → 더 많은 것을 얻는다 | 비전에 순수하게 집중 → 예상 밖 기회 열림 |
| 취약해지면 → 더 강해진다 | 숨은 의도 드러냄 → 그 힘이 약화됨 |
