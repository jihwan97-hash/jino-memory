# Learnings / Lessons

## [LRN-20260304-001] 설치 전 검증 먼저
**상황:** memory-lancedb, agent-memory-ultimate를 검증 없이 설치했다가 결국 제거
**교훈:**
- 새 스킬/플러그인 설치 전에 반드시 "이게 진짜 필요해?" 먼저 판단
- 기존 시스템과 역할 겹치는지 먼저 체크
- 설치 후 실제 작동 검증까지 못 하면 "검증 안 됨"이라고 솔직하게 말하기
- 불필요하거나 검증 안 된 레이어는 추가하지 말 것

**체크리스트 (설치 전):**
1. 기존 시스템에서 이미 커버되는 기능인가?
2. 실제로 작동하는지 빠르게 검증 가능한가?
3. 설치 후 롤백이 쉬운가?
4. 없어도 지금 잘 돌아가고 있는가? → 없어도 됨
